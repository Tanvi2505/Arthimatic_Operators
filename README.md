# Arithmetic Operators in Java

This repository contains Java programs that demonstrate the use of **Arithmetic Operators**. These operators are used to perform basic mathematical operations such as addition, subtraction, multiplication, division, and modulus.

## 🔹 Operators Covered

- `+` → Addition  
- `-` → Subtraction  
- `*` → Multiplication  
- `/` → Division  
- `%` → Modulus (remainder)

## 🔹 Description

The programs in this repository help beginners understand how arithmetic operations work in Java. Each program is simple and easy to follow, making it useful for students and beginners learning Java fundamentals.

## 🔹 Features

- Simple and clean code
- Beginner-friendly examples
- Covers all basic arithmetic operations
- Helps understand operator behavior

## 🔹 Example

```java
int a = 10;
int b = 5;

System.out.println("Addition: " + (a + b));
System.out.println("Subtraction: " + (a - b));
System.out.println("Multiplication: " + (a * b));
System.out.println("Division: " + (a / b));
System.out.println("Modulus: " + (a % b));
